# Solar Selector

A modern React application built with:
- **Vite** (Next Generation Frontend Tooling)
- **React 19**
- **Tailwind CSS v4** (Zero-config, CSS-first optimization)

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start development server**:
    ```bash
    npm run dev
    ```

3.  **Build for production**:
    ```bash
    npm run build
    ```

## Project Structure

- `src/main.jsx`: Entry point for the React application.
- `src/App.jsx`: Main application component with premium styling.
- `src/index.css`: CSS entry point with Tailwind 4 integration and custom theme variables.
- `vite.config.js`: Configuration for Vite, React, and Tailwind 4.

## Tailwind CSS v4

This project uses Tailwind CSS v4. Notable features:
- No `tailwind.config.js` needed by default.
- Custom theme variables are defined in `src/index.css` using the `@theme` rule.
- High-performance, Lightning CSS-powered compilation.
